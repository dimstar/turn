# A Nextjs, Typescript Example Project

Start the development server.

```bash
  ~$ yarn dev
```

Build the app for production.

```bash
  ~$ yarn build
```

Run the built app in production mode.

```bash
  ~$ yarn start
```

Check types

```bash
  ~$ yarn check
```
